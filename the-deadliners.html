About the Game

The Deadliners is a top-down survival game where the arena is never the same twice. Each run generates a new maze using BFS/DFS algorithms, creating a fresh layout every time. The goal is simple: find the exit. But enemies use A* pathfinding to hunt the player through the same corridors they're trying to escape.

The game has two modes. Easy mode is a clean exploration challenge — no enemies, just solving the maze at your own pace. Hard mode switches on the enemy AI, turning navigation into an urgent chase where every dead end is a potential trap.

Built in Pygame with a 3-person team, The Deadliners was a focused technical project centered on combining procedural generation with real-time pathfinding to make a maze that's both unique and dangerous every playthrough.

My Contributions

Procedural Maze Generation (BFS/DFS) Built the maze generation system using BFS and DFS algorithms. The system carves a unique, fully-connected maze layout on every run — no two playthroughs share the same path structure. Responsible for choosing and implementing the traversal logic that ensures every generated maze is solvable.
Game Setup & Project Foundation Set up the overall game structure in Pygame — scene management, game loop, rendering pipeline, and the integration layer that connects maze generation with player movement and the enemy AI system.
