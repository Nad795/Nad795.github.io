About the Game

Boombatag 2 is a co-op bomb-defusal puzzle game played in teams. Each team gets a suitcase with a unique Serial Code — a key that determines which version of each puzzle they face. The suitcase has 6 modules to defuse: 6 puzzles and 4 problems, all within a 2-hour time limit.

Every wrong answer lights up a red lamp on the suitcase. Five lamps means the bomb explodes and the game ends — so accuracy matters as much as speed. Modules range from Magic Square (arrange 16 numbers so every row, column, and diagonal sums to N) and Dice Puzzle (group a 9×9 grid correctly) to Wire Cutting (a decision tree for slicing the right cable) and Wheel of Confusion (a spinning wheel that must point to the answer of a math problem). A Passkey module rotates the correct answer every 90 seconds based on each team's ID, adding time pressure to an already intense experience.

Built in Unity with a team of 10 for Schematics NLC 2024, the game is designed around communication, coordination, and keeping cool under pressure.

My Contributions

Wheel of Confusion Module Built the full Wheel of Confusion puzzle module — a spinning wheel mechanic where players click rotation direction buttons and a step-count button to spin the wheel to the correct answer for a math problem. The correct problem set is determined by the team's Serial Code (number of digits, whether it starts with an even/odd digit, etc.), and the team gets two chances before a mistake is counted.
